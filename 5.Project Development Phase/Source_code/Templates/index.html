Visualizing Housing Market Trends: An Analysis of Sales Price and Feature using Tableau

This project focuses on analyzing housing market data through interactive visualizations. The goal is to explore how house prices are distributed and understand renovation trends within the dataset. Using modern data visualization techniques, the project transforms raw housing data into clear and meaningful insights.

The dashboard was developed using Tableau to visualize key patterns such as sale price distribution, price ranges, and renovation timelines. By presenting the information through charts and filters, users can easily explore the dataset and identify trends that influence housing values.

This interactive dashboard demonstrates how data visualization can help simplify complex housing data and reveal valuable market insights.

  • Interactive histogram showing the distribution of house sale prices.
  • Analysis of renovation timelines and their relationship to housing prices.
  • Dynamic filters allowing users to explore price ranges and dataset patterns.

The project website is built using Bootstrap to provide a clean and responsive interface for presenting the Tableau dashboard. It combines web development and data visualization to deliver an engaging and informative data analysis experience.
